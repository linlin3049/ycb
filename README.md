# 新生儿遗传病推测工具

基于孟德尔遗传定律，输入家庭成员的患病状态后：

- 推测父亲、母亲、爷爷、叔叔、姨姨的基因型概率；
- 自动计算新生儿基因型概率。

> 当前支持 **常染色体隐性（AR）** 与 **常染色体显性（AD）** 两类模型。

## 使用方法

```bash
python3 neonatal_genetics.py
```

按提示输入：

1. 疾病名称（内置疾病会自动选择遗传模式和默认致病等位基因频率）；
2. 父亲、母亲、爷爷、叔叔、姨姨是否患病（可输入 `患病/未患病`、`是/否`、`1/0`）；
3. 若疾病不在内置库中，可手动输入遗传模式与等位基因频率。

## 内置疾病示例

- 囊性纤维化（AR）
- 苯丙酮尿症（AR）
- 地中海贫血（AR）
- 亨廷顿舞蹈症（AD）
- 软骨发育不全（AD）

## 注意

- 这是教学/科普计算模型，不可替代医学诊断与遗传咨询。
- 模型假设完全外显率、无新发突变、无测量误差。
